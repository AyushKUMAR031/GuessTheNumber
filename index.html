Guess The Number Game

Welcome To The Arena

    - Game Name: Guess the Correct Number
    - Objective: Players attempt to guess a secret number within a specified range.
    - Rules:
        The game host selects a random number between a minimum and maximum value (e.g., between 0 and 100).
        Players take turns guessing the secret number.After each guess, the host provides feedback:
          1) If the guessed number is correct, the player wins.
          2) If the guessed number is too high, the host says, “Lower!”
          3) If the guessed number is too low, the host says, “Higher!”
    - Players continue guessing until someone correctly identifies the secret number or until a predetermined number of attempts is reached.
    - The player with the correct guess wins the round.
